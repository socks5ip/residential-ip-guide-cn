# 贡献指南（Contributing Guide）

感谢您关注本仓库！欢迎通过 Issue 或 Pull Request 贡献。

## 如何贡献

- 报告链接失效
- 新增住宅IP提供商（需支持免费测试、真实住宅IP）
- 修正错误内容
- 补充教程或技巧

## 贡献流程

### 通过 Issue
点击 [Issues](https://github.com/socks5ip/residential-ip-guide-cn/issues) → New Issue → 选择模板 → 提交。

### 通过 Pull Request
1. Fork 本仓库
2. 克隆到本地
3. 创建新分支
4. 修改文件（保持 Markdown 格式正确）
5. 提交并推送
6. 创建 Pull Request

## 格式要求

- 标题使用 `#`、`##`
- 表格使用 `| --- |`
- 链接使用 `[文字](URL)`
- 代码块使用 ` ``` `

## 行为准则

- 友好交流
- 内容真实合法
- 禁止广告或垃圾信息

**感谢您的贡献！**
