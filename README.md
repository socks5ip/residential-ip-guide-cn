## 🔍 住宅IP速览：什么是住宅IP？

**住宅IP（Residential IP）** 是由互联网服务提供商（ISP，如中国电信、Comcast、AT&T等）分配给真实家庭宽带的IP地址，与数据中心/机房IP有着本质区别：

| 对比项 | ✅ 住宅IP | ❌ 机房IP |
| --- | --- | --- |
| 来源 | 家庭宽带运营商 | IDC机房、云服务商 |
| ASN类型 | ISP / Residential | Hosting / Datacenter |
| 行为特征 | 像真实用户访问 | 偏向脚本/程序请求 |
| 风控容忍度 | 高，难识别 | 低，易封禁 |
| 连号情况 | 分散、不连号 | 容易连号，纯净度差 |

> 一句话总结：**住宅IP看起来像真实用户，机房IP更像自动化程序**。

如何快速判断一个IP是否为真正的住宅IP？
- 通过 WHOIS/IP 查询工具查看 ASN 归属类型——显示为 `Residential` 或 `ISP` 的是住宅IP，显示 `Hosting` 或 `Datacenter` 的是机房IP
- 查看反向 DNS/PTR 记录和端口暴露情况（住宅IP通常不对外暴露 22/80/443 等端口）
- 更详细的检测方法请访问：**[IP质量检测中心](https://socks5ip.com.cn/ip-check)**（专业IP地址查询 · 住宅IP/机房IP识别 · 代理/VPN识别）
